# Thư mục Images

## Logo
File `logo.png` là file placeholder. Vui lòng thay thế bằng logo thực tế của bạn.

### Yêu cầu logo:
- Định dạng: PNG (khuyến nghị) hoặc SVG
- Kích thước: Tối thiểu 200x200px
- Nền: Trong suốt (transparent) hoặc nền tối
- Tỷ lệ: Hình vuông hoặc gần vuông

### Cách thay thế:
1. Đặt file logo mới vào thư mục này với tên `logo.png`
2. Đảm bảo file có kích thước phù hợp
3. Refresh trang web để xem logo mới
